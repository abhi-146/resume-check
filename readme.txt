Display feeds from Threads, an Instagram app on your website.

== Description ==

Best WordPress threads plugin that allows admin to display feeds from Threads, an Instagram app on your website. No need to enter any API key to load posts from threads.

You just need the username of a public profile on Threads and the plugin does the rest for you.

== Installation ==

You can install the Plugin in two ways.

= WordPress interface installation =

1. Go to plugins in the WordPress admin and click on “Add new”.
2. In the Search box enter “feeds for threads” and press Enter.
3. Click on “Install” to install the plugin.
4. Activate the plugin.

= Manual installation =

1. Download and upload the plugin files to the /wp-content/plugins/user-private-files directory from the WordPress plugin repository.
2. Activate the plugin through the "Plugins" screen in WordPress admin area.

== Frequently Asked Questions ==

= Do I need API? =

No, this plugin works without forcing you to have API keys.

= What are the shortcodes =
Use Shortcode [fftmj_feeds] to load threads from user you set in settings.
Or use [fftmj_feeds username="threads_username_here"] to load threads from different a user. Replace "threads_username_here" with your username.
